## üîç Troubleshooting

### Common Issues

#### 1. "Microsoft Visual C++ 14.0 required"
**Solution:** Use `install_quick_windows.ps1` which uses pre-built wheels

#### 2. sounddevice/PortAudio error
**Solution:** Voice interface works without audio I/O (text-only mode)

#### 3. TensorFlow warnings
**Solution:** Ignore "oneDNN" and "TF-TRT" warnings - these are informational

#### 4. Flower/grpcio conflicts
**Solution:** Let flwr manage its own grpcio version (don't pin manually)

#### 5. Azure/GCP not available
**Solution:** Optional - CHIMERA works with AWS only or no cloud at all

### Full Troubleshooting Guide
